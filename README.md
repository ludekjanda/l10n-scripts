# l10n-scripts
Bash scripts to automate and streamline localization



Credentials are stored in a separate ~/.l10nrc file

```
export DEEPL_AUTH_KEY=[DEEPL AUTHENTICATION TOKEN]
export SB_AUTH_KEY=[STORYBLOK AUTHENTICATION TOKEN]
```

Phrase credentials are stored in a separate ~/.phraserc file

```
export PHRASE_USERNAME="[PHRASE USERNAME]"
export PHRASE_PASSWORD="[PHRASE PASSWORD]"
export PHRASE_TOKEN=[LEAVE EMPTY, IT WILL PRE_FILL ITSELF]
```
