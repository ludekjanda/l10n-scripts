# l10n-scripts
Bash scripts to automate and streamline localization

Credentials are stored in a separate ~/.l10nrc file

```
export DEEPL_AUTH_KEY=[STORYBLOK AUTHENTICATION TOKEN]
export SB_AUTH_KEY=[STORYBLOK AUTHENTICATION TOKEN]
```



## l10n-deepl.sh
translates documents with DeepL 

## l10n-sb.sh
translates Storyblok articles with DeepL
